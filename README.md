## Tech Stack

**Client:** React, Redux, SCSS, Material UI, 

**Server:** Node, Express, MongoDB, Firebase, Axios API


## Features

- Fully functional and dynamic with modern React Hooks, Redux implementation, Node.js/Express.js server, with REST API, Database integration, Login & Register user authentication with JWT.
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
